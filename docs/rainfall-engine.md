# Rainfall Engine

The rainfall engine computes design rainfall intensity.

## Data Sources

- IMD gridded rainfall datasets
- Rain gauge station data
- Rainfall zonal maps

Typical rainfall time series span 30–100 years.

---

## Rainfall Frequency Analysis

Extreme rainfall events are estimated using:

- Log Pearson Type III distribution
- Best fit curves

Process:

Rainfall time series → statistical fitting → rainfall for return period
