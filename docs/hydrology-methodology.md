# Hydrology Methodology

## Design Input

Hydrology analysis begins with a crossing location.

Example:

Bridge or culvert at coordinate X.

The goal is to estimate:

- Design flood discharge
- High Flood Level (HFL)

---

## Return Period Selection

Return period depends on infrastructure type.

| Structure | Return Period |
|-----------|--------------|
Drain | 10–25 years |
Culvert | 25–50 years |
Bridge | 50–100 years |

---

## Catchment Analysis

Using DEM terrain models the system extracts:

- Catchment boundary
- Catchment area
- Drainage network
- Slope characteristics
