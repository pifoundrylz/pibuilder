# Product Overview

## Problem

EPC companies designing bridges, culverts, and drainage structures must estimate:

- Flood discharge
- High Flood Level (HFL)

Today this workflow is manual and fragmented across multiple tools:

- IMD rainfall datasets
- GIS software for watershed analysis
- Excel sheets for hydrology calculations
- Engineering code tables for runoff coefficients

This makes hydrology analysis slow, inconsistent, and difficult to scale.

---

## Vision

Build a **Hydrology Decision Platform** where:

Location → Catchment → Rainfall → Runoff → Discharge → HFL → Report

Engineers simply select a point on the map and receive a complete hydrology analysis.

---

## Target Users

Primary users include:

- EPC contractors
- Hydrology consultants
- Infrastructure design firms
- Government agencies reviewing DPRs
